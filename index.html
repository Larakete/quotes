Hello and welcome everybody

Glad that you are here and are interested in contributing to our beginnerfriendly webproject. Our intention is to keep this project as simple as possible and to let everyone contribute who wants to. As we want to have a valueable output to give something to the beautiful open source community, we tried to devide this projects into some useful subprojects.

Important: If there is anythink you want to add to this page - feel free. It is a great way to learn how to contribute to open source and it is the first step to go deeper in contributing to open source.

The following subprojects are implemented (at the moment). At the beginning of the description we added what you will learn by accomplishing the task.

  • HTML: Ask and answer what you like in the context of programming
  • HTML/JAVASCRIPT: We want to build a random quote generator
